### Read Me

First, install node using:

    brew install nvm
    nvm install 0.10.32

Then, setup node-inspector:

    npm install -g node-inspector

To install:

    npm install

To analyze your code:

    grunt jshint

To run the webserver:

    grunt dev

To run the test suite:

    grunt test

To debug the test suite:

    ./dbst
